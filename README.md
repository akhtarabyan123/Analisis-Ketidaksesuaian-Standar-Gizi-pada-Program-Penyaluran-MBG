# Analisis-Ketidaksesuaian-Standar-Gizi-pada-Program-Penyaluran-MBG
Program MBG yang semakin masif masih menghadapi masalah kesesuaian standar gizi dan kelayakan makanan. Masih ditemukan kasus makanan basi hingga keracunan siswa, sementara kandungan gizi belum sesuai dengan usia sasaran program. Akibatnya, tujuan utama MBG untuk meningkatkan kualitas gizi anak belum tercapai secara optimal.
